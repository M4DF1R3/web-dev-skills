WEB DEV SKILLS

Learn HTML

HTML stands for Hyper Text Markup Language. HTML is the content of a web page.
Examples of HTML include heading, paragraphs, images, links, etc.

Check out the HTML Tutorial on w3schools.com

Learn CSS

CSS stands for Cascading Style Sheets. CSS is the style of a web page.
Examples of CSS include font styles and colour, background colour, borders
dimensinos, layouts, etc.

Check out the CSS Tutorial on w3schools.com

Learn JavaScript

JavaScript is the programming language of the web. JavaScript helps to make
web pages interactive and dynamic. Some commons JavaScript tasks are:

  • add new HTML to the pages
  • change the existing content/style
  • react to user events(mouse clicks, key presses, etc.)

Check out JavaScript Tutorial on javascript.info
